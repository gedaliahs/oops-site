FAQ

Common questions about oops.

Can you undo rm in the terminal?
Not by default — rm permanently deletes files. But if you have oops installed, it automatically backs up files before rm runs. Type oops to restore them.
Can you undo rm -rf?
Without a backup tool, no. rm -rf permanently deletes files and directories. oops catches rm -rf before it runs, backs up the entire directory tree using hard links (instant, no extra disk space), and lets you restore with one command.
Is there a Ctrl+Z for the terminal?
Not built into any shell. oops adds this by installing a shell hook that watches for destructive commands and lets you undo by typing oops.
How do I undo git reset --hard?
Without oops: check git reflog for committed changes, but uncommitted work is usually gone. With oops: it creates a git stash before the reset. Type oops to apply the stash and get your changes back.
How do I recover a file after sed -i?
sed -i edits files in place with no backup. If the file is in git, try git checkout. If oops is installed, it copies the file before sed -i runs, and you can restore with oops.
Does oops slow down my terminal?
No. Non-destructive commands have zero overhead — no subprocess is spawned. Destructive commands take about 10ms to back up. You won't notice it.
How much disk space does oops use?
Minimal. Backups use hard links, which don't use extra space until the original is deleted. Auto-cleanup removes entries older than 7 days, with a 5GB cap. Both are configurable via oops config.
Does oops work with AI coding agents?
Yes. Claude Code, Cursor, Aider, Codex — any tool that runs shell commands goes through the same hook. If an agent runs rm or git reset --hard, oops catches it.
What shells does oops support?
zsh, bash, and fish. It uses each shell's native preexec mechanism.
Does oops work on Linux?
Yes. macOS and Linux, amd64 and arm64. Single Go binary, no dependencies.
What commands does oops catch?
rm, rm -rf, mv (overwrites), sed -i, chmod, chown, git reset --hard, git checkout ., git branch -D, git clean -fd, file redirects (> file), and truncate.
How do I uninstall oops?
Run oops uninstall. It removes the shell hook and backup directory. Then run sudo rm /usr/local/bin/oops to remove the binary.
